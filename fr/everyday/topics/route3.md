### 🔄 Register-flip — les six dialogues

👉 Consigne : lis d'abord la version **standard**, puis la version **familière** (parlée). Repère ce qui change : le _ne_ qui tombe, « il y a » → « y'a », la dislocation (« Il est vert, **le feu**… »), et les tournures idiomatiques (_t'inquiète_, _faire gaffe_, _y'a pas d'souci_).

> <span onclick="playAudio('dialog1')">**💬 Dialogue 1 — Piéton ↔ Conducteur (au passage piéton)**</span>
>
> **Standard**<br>
> Piéton : Excusez-moi, je peux traverser ?<br>
> Conducteur : Oui, allez-y.<br>
> Piéton : Merci beaucoup.<br>
> Conducteur : Pas de problème.
>
> **Familier**<br>
> Piéton : Pardon, j'peux traverser ?<br>
> Conducteur : Ouais, allez-y.<br>
> Piéton : Merci !<br>
> Conducteur : Y'a pas d'souci.

> <span onclick="playAudio('dialog2')">**💬 Dialogue 2 — Piéton ↔ Conducteur (feu piéton)**</span>
>
> **Standard**<br>
> Piéton : Le feu est vert pour les piétons.<br>
> Conducteur : Oui, je m'arrête.<br>
> Piéton : Merci.<br>
> Conducteur : Bonne journée.
>
> **Familier**<br>
> Piéton : Il est vert, là, pour nous.<br>
> Conducteur : Ouais, j'm'arrête.<br>
> Piéton : Merci !<br>
> Conducteur : Bonne journée.

> <span onclick="playAudio('dialog3')">**💬 Dialogue 3 — Piéton ↔ Cycliste (attention !)**</span>
>
> **Standard**<br>
> Piéton : Attention !<br>
> Cycliste : Oh, pardon !<br>
> Piéton : Pas de souci.<br>
> Cycliste : Merci.
>
> **Familier**<br>
> Piéton : Attention !<br>
> Cycliste : Oh, pardon !<br>
> Piéton : T'inquiète.<br>
> Cycliste : Merci.

> <span onclick="playAudio('dialog4')">**💬 Dialogue 4 — Piéton ↔ Cycliste (piste cyclable)**</span>
>
> **Standard**<br>
> Piéton : Oups, désolé, je n'ai pas vu la piste cyclable.<br>
> Cycliste : Ce n'est rien.<br>
> Piéton : Je fais attention.<br>
> Cycliste : Merci.
>
> **Familier**<br>
> Piéton : Oups, désolé, j'avais pas vu la piste.<br>
> Cycliste : C'est rien.<br>
> Piéton : J'fais gaffe.<br>
> Cycliste : Merci.

> <span onclick="playAudio('dialog5')">**💬 Dialogue 5 — Conducteur ↔ Cycliste**</span>
>
> **Standard**<br>
> Conducteur : Vas-y, je te laisse passer.<br>
> Cycliste : Merci !<br>
> Conducteur : Pas de problème.
>
> **Familier**<br>
> Conducteur : Vas-y, j'te laisse passer.<br>
> Cycliste : Merci !<br>
> Conducteur : Y'a pas d'quoi.

> <span onclick="playAudio('dialog6')">**💬 Dialogue 6 — Cycliste ↔ Conducteur (presque une collision)**</span>
>
> **Standard**<br>
> Cycliste : Le feu est rouge pour les voitures !<br>
> Conducteur : Ah oui, désolé.<br>
> Cycliste : Fais attention.<br>
> Conducteur : Oui, excuse-moi.
>
> **Familier**<br>
> Cycliste : Il est rouge, le feu, pour les voitures !<br>
> Conducteur : Ah ouais, désolé.<br>
